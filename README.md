# ssrmu2

A lightweight compatibility launcher for running the classic `ssrmu.sh` menu on newer Linux systems.

It keeps the original interactive menu and only prepares the basic compatibility environment needed by modern distributions.

## Usage

```bash
wget --no-check-certificate -O ssrmu-modern.sh https://raw.githubusercontent.com/SSTAPAPP/ssrmu2/main/ssrmu-modern.sh
chmod +x ssrmu-modern.sh
bash ssrmu-modern.sh
```

After startup, follow the on-screen menu.
